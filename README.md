2. Select your columns to convert, you can select all columns.

<h3 align="left">
    Convert CSV to Excel
    <br>
</h3>

1. Upload your `csv` file
2. Select your columns to convert, you can select all columns.
3. Click on the button [**convert to xlsx file**]  üöÄ
